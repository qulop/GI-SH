1. write
	Создать запись о расходах за день.
		Для начала ввода: <write>

		Для создания поля: <[spending] [spent] %flags%>
			%flags% - категория расходов:
				1) -imp - важное
                                2) -ent - развлечение
                                3) -transp - транспорт
                                4) -food - еда
                                5) -house - комунальные услуги
                                6) -credit - кредит
                                7) -vehicle - трата на личное транспортное средство

				
		Для исправлления поля: <correct [lineno]>
			Например:
                                user-$write$
                                        line-1: mcdonalds 12 -food     <- bad.

                                user-$write$
                                        line-2: change 1

                                user-$write$
                                        line-1: vegetables 2 -food    <- good.	

		Для смены валюты: <currency %flags%>
			%flags% - название вылюты. По умолчанию -usd.
                       Поддерживаемые по умолчанию валюты:
                                1) -mnt - монгольский Тугрик
                                2) -usd - американский Доллар
                                3) -eur - Евро
                                3) -rub - российские Рубли
                                4) -byn - белорусские Рубли
                                5) -uah - украинская Гривна
                                6) -czk - чешская Крона
                                7) -pln - польский Злотый
                                8) -gbn - британский Фунт-Стерлингов
                                9) -jpy - японская Йена
                                10) -cny - китайский Юань
                                11) -ils - израильский Шекель
                                12) -krw - корейская Вона
                                13) -kzt - казахский Тенге
                                18) -aud - австралийский Доллар
                                19) -cad - канадский Доллар
                                20) -btc - Биткоин
                                21) -eth - Эфириум
                                22) -doge - Dogcoin
                                23) -usdt - Tether

                Для добавления новой категории/валюты: <add [flagname] %flags%>
                        [flagname] - новая валюта/категория:
                                Длинна флага для:
                                        1) Категории - не более 7-ми символов
                                        2) валюта - не более 4-х символов
                        %flags%:
                                1) -cur - для добовления новой валюты
                                2) -cat - для добавления новой категории

3. see
        Просмотреть статистику расходов за указанный промежуток времени/за конкретный день. 
        Если промежуток не указан, статистика будет отображена за все время пользования утилитой.
                Синтаксис: <see ?([from] [to])? ?[day]? ?%flags%?>
                	1) ([from] [to]) - Просмотр статистики за указанный промежуток
                	2) [day] - За конкретный день
                	3) %flags%:
                		3.1) -table(по умолчанию) - отобразить статистику расходов в виде таблицы
                		3.2) -text - отобразить статистику расходов в текстовом виде


4. chappear
        Изменить параметры отображения даты в консоли.
                Синтаксис: <chappear [type] ?%flags%?>
                [type]:
                        1) YMD  ->  user-$2022.03.26$
                        2) DMY  ->  user-$26.03.2022$

                %flags%:
                        1) -text  ->  user-$2022 march 26$  or  user-$26 march 2022$
                        2) -num  <->  YMD  or  DMY


5. help
        Узнать синтаксис и назначение каждой команды.
                Синтаксис: <help ?[cmdname]? ?%flags%?>
                %flags%:
                        1) -signs - узнать расшифровку синтаксиса, используемого в подсказках.


6. help -signs
        синтаксис подсказок:
                1) <> - обозначение команды для удобочитаемости подсказки
                2) [] - место командлета команды
                3) : type - тип конмандлета
                4) () - группировка командлетов, в вызове команды обязаны идти вместе.
                4) ?? - необязательная часть команды
                5) %flags% - обозначение флагов
        Переменные и Функции:
                1)  Переменная - это именнованный "контейнер", в котором храниться определенная информация.
                    Переменные нацелены на упрощение пользования командной строкой.
                    Например, переменная $today хранит информацию о сегодняшней дате.
                2)  Функции - ...
                    Функция jump(n: int) принимает на вход один аргумент типа integer и испльзуется для
                    Перемещение на n-дней назад.


7. alias
        Создать синонимы для ключевых слов. После создания синонима вы можете использовать как оригинальное ключевое слово, так и созданный синоним
                Для начала: <alias>
                Для создания синонима: <[replacement statement] [original statement]]>
                Для возвращения на .. строку: <correct [lineno: int]>
		        user-$alias$
		                line-1: alias tulpae     <- bad.

		        user-$alias$
		                line-2: change 1

		        user-$alias$
		                line-1: tulpae alias     <- good.


8. correct
        Добавить/изменить/удалить запись в статистике расходов за указанный день.
                Синтаксис: <correct [day] ?[spendname]? ?[lineno]? ?([line] [column])?>
                [day] - День, в записи которого вы хотите что-либо изменить.
                        
                        По умолчанию: <correct $today>
                Опциональные вырианты(допускается использование только один из них):
                        1) [spendname] - название расхода
                        2) [lineno] - используется в купе с командой seestat при использовании флага -text
                        3) ([line] [column]) - используется в купе с командой seestat при использовании флага -table


9. settings
	Выведется информация с пользовательскими настройками утилиты.
			Синтаксис: <settings>


10. hint
	Управление отображения подсказок в сообщениях об ошибках или сообщениях с предупреждениями.
			Синтаксис: <hint %flags%>
			%flags%:
					1) -on (Используется по умолчанию):
							Warning: unknown command -> "unknown_comand"
							to change this line(12) use: <change 12>	<- hint
					2) -off:
							Warning: unknown command -> "unknown_comand"
